# SeeATheater V1.0

## 📖 프로젝트 소개
**CC(씨씨)**는 "씨어씨어터(See Theater)"의 줄임말로, **"See See! - 직접 보고, 즐기자"**는 의미를 담은 캠퍼스 공연 전용 플랫폼입니다.

학교 연극제부터 독립 소극장, 뮤지컬 동아리 공연까지 캠퍼스 안팎의 다채로운 공연 정보를 한 눈에 모아보고, 간편하게 예매할 수 있는 원스톱 서비스를 제공합니다.

🎯 서비스 목표
🏫 캠퍼스 문화 활성화: 대학가 공연 문화의 접근성을 높여 더 많은 학생들이 참여할 수 있도록 지원

🎪 공연 정보 통합: 흩어져 있던 공연 정보를 하나의 플랫폼에서 통합 관리

🎟️ 간편한 예매 시스템: 복잡한 예매 과정을 간소화하여 사용자 편의성 극대화

👥 커뮤니티 형성: 공연을 사랑하는 사람들 간의 소통과 정보 공유 공간 제공
